# Root cause:
src/utils/ssrf.ts — safeLookup() callback signature mismatch with undici v6+ Agent.connect.lookup contract.
Current code: cb(null, pick.address, pick.family)  — tri-arg style (dns.lookup all:false)
Undici expects: cb(null, [{address, family}, ...])  — addresses array
