How to query (L402)

1
Request
Call any scored endpoint without credentials.
GET /api/agent/{hash}
2
Receive invoice
Server returns HTTP 402 with a Lightning invoice in the WWW-Authenticate header. Cost: 1 sat.
3
Pay & retry
Pay the invoice, then resend your request with the L402 token to get the score.
# Step 1 — Request (returns 402 + invoice)
curl -i https://satrank.dev/api/agent/{hash}
HTTP/2 402
# WWW-Authenticate: L402 macaroon="AGI...", invoice="lnbc10n1..."
# Step 2 — Pay the invoice (1 sat)
lncli payinvoice lnbc10n1...
# Step 3 — Retry with L402 token
curl -H "Authorization: L402 <macaroon>:<preimage>" https://satrank.dev/api/agent/{hash}

Agent Decision API

The 3 endpoints an autonomous agent needs. Decide before paying, report the outcome, query the profile. How it works →

  • POST /api/decide GO / NO-GO with success probability 1 sat
  • POST /api/report Report transaction outcome free
  • GET /api/profile/:id Full agent profile with reports & uptime 1 sat

Discovery free

  • GET /api/agents/top Leaderboard by score
  • GET /api/agents/movers Top movers (7-day delta)
  • GET /api/agents/search?alias=... Search by alias
  • GET /api/stats Network statistics

Detailed Queries 1 sat

  • GET /api/agent/:hash Detailed score & evidence
  • GET /api/agent/:hash/verdict SAFE / RISKY / UNKNOWN
  • GET /api/agent/:hash/history Score history
  • POST /api/verdicts Batch verdicts (up to 100)

Tools
