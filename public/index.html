Live on Nostr

SatRank publishes trust assertions for the Lightning nodes it tracks as NIP-85 events (kind 30382) every 6 hours on public Nostr relays. ~5,000 assertions maintained on relays (score ≥ 30). Any Nostr-native agent or client can consume scores without touching our API. Live node count and stats are at /api/stats.

This is the only NIP-85 provider scoring the Lightning payment graph. Every other provider scores the Nostr social graph (follows, mutes, zaps). SatRank scores who you can pay.

  • NIP-85 kind 30382 Trusted assertions — score, verdict, components, reachability
  • NIP-90 kind 5900 → 6900 DVM trust-check — real-time score for any Lightning node
  • NIP-05 satrank@satrank.dev Verified Nostr identity

Relays: relay.damus.io · nos.lol · relay.primal.net

Agent Decision API

The full pipeline: screen, route, decide, report, profile. How it works →

  • POST /api/verdicts Batch screen up to 100 targets 1 req
  • POST /api/best-route Parallel pathfinding, top 3 by route quality 1 req
  • POST /api/decide GO / NO-GO with success probability + positional pathfinding 1 req
  • POST /api/report Report transaction outcome free
  • GET /api/profile/:id Full agent profile with reports & uptime 1 req

Discovery free

  • GET /api/ping/:pubkey Real-time reachability check (QueryRoutes live)

  • GET /api/stats Network statistics
