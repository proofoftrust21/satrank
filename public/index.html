How to query (L402)

1
Request
Call any scored endpoint without credentials.
GET /api/v1/agent/{hash}
2
Receive invoice
Server returns HTTP 402 with a Lightning invoice in the WWW-Authenticate header. Cost: 1 sat.
3
Pay & retry
Pay the invoice, then resend your request with the L402 token to get the score.
# Step 1 — Request (returns 402 + invoice)
curl -i https://satrank.dev/api/v1/agent/{hash}
HTTP/2 402
# WWW-Authenticate: L402 macaroon="AGI...", invoice="lnbc10n1..."
# Step 2 — Pay the invoice (1 sat)
lncli payinvoice lnbc10n1...
# Step 3 — Retry with L402 token
curl -H "Authorization: L402 <macaroon>:<preimage>" https://satrank.dev/api/v1/agent/{hash}

API Endpoints

Scores are computed from 5 weighted components. The Verdict API returns SAFE/RISKY/UNKNOWN for <200ms decisions. Full methodology →

  • POST /api/v2/decide GO / NO-GO with success probability v2
  • POST /api/v2/report Report outcome (free) v2
  • GET /api/v2/profile/:id Agent profile with reports & uptime v2
  • GET /api/v1/agent/:hash/verdict SAFE / RISKY / UNKNOWN verdict
  • POST /api/v1/verdicts Batch verdicts (up to 100 hashes)
  • GET /api/v1/agent/:hash Detailed agent score
  • GET /api/v1/agent/:hash/history Score history
  • GET /api/v1/agent/:hash/attestations Received attestations
  • GET /api/v1/agents/top Leaderboard by score
  • GET /api/v1/agents/movers Top movers (7-day delta)
  • GET /api/v1/agents/search?alias=... Search by alias
  • POST /api/v1/attestations Submit an attestation
  • GET /api/v1/health Service status
  • GET /api/v1/stats Network statistics
  • GET /api/v1/version Build commit and date
  • GET /api/v1/openapi.json OpenAPI specification
  • GET /api/v1/docs Interactive API explorer (Swagger UI)
